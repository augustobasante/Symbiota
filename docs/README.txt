SOBRE ESTA PLATAFORMA
===================

El Proyecto de Software Symbiota está construyendo una biblioteca de herramientas web para
ayudar a los biólogos en el establecimiento de espécimen de floras virtuales y
faunas. Este proyecto desarrollado a partir de la realización de ese complejo,
Los portales de biodiversidad ricos en información se construyen mejor a través de
Esfuerzos colaborativos entre desarrolladores de software, biólogos,
Gestores de vida silvestre, y científico ciudadano. La premisa central
De este proyecto de software de código abierto es que a través de una asociación
Entre ingenieros de software y comunidad científica, mayor calidad.
y se pueden construir portales de biodiversidad más útiles para el público. Un
El marco de software de código abierto permite a los técnicos crear
Las herramientas, liberando así al biólogo para concentrar sus esfuerzos.
En la curación de conjuntos de datos de calidad. En esta mansión, podemos crear.
algo mucho mayor que una sola entidad es capaz de hacer en
su propia

Se puede acceder a más información sobre este proyecto a través de:

http://symbiota.org/

NOTAS
=====


CARACTERISTICAS
========

* Motor de búsqueda de muestras
   * Tesauro taxonómico para consultar sinónimos taxonómicos.
   * Google Map y capacidades de mapeo de Google Earth
   * Lista dinámica de especies generada a partir de registros de especímenes.
* Sistema de manejo de flora / fauna.
   * Lista de especies estáticas (floras / faunas locales)
* Claves de identificación interactivas
   * Generación de claves para la lista de especies dentro del sistema.
   * Generador de claves basado en una localidad puntual.
* Biblioteca de imágenes


LIMITACIONES
===========

* Solo probado en sistemas operativos Linux y Windows (XP, Vista)
* Debería funcionar con un servidor web habilitado para PHP aunque solo
  probado con el servidor HTTP Apache
